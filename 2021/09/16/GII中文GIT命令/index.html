GII中文GIT命令

本文最后更新于：2023年4月4日 晚上

前言

GII 是基于 C++ 开发的 GIT 中文命令程序，安装后可以使用中文命令替代 GIT 命令，首发1.0版本包含了大量常用的 GIT 命令，支持 Windows 和 Linux 操作系统。GII 依赖于 GIT，使用前需要先安装 GIT。

更新

  • 2021年10月6日：

    • 更新 gii v1.0.1
    • 优化修改大量代码

  • 2021年9月16日：

    • 发布首个版本 gii v1.0 包括 Windows 和 Linux 的两个版本

安装：

Windows：

  • 点击下载 GII
  • 将 gii.exe 拷贝到 GIT 安装目录下的 cmd 目录
  • 推荐使用 CMD 命令提示符来操作，可以更好的支持中文编码

使用说明：

中英文命令对照表：

中文命令 英文命令 说明
gii 帮助 gii -h 显示帮助信息
gii 版本 gii -v 显示 gii 版本
gii 状态 git status 显示工作区状态
gii 显示分支 git branch 显示本地分支
gii 显示远程分支 git branch -r 显示远程分支
gii 显示所有分支 git branch -a 显示本地和远程所有分支
gii 日志 git log 显示仓库日志
gii 入库日志 git log –graph –pretty=format:”%h %s” now 显示仓库日志(只显示 id 和版本)
gii 退库日志 git reflog 显示回退掉的仓库日志
gii 初始化 git init 初始化仓库
gii 克隆 地址 git clone 地址 克隆仓库到一个新目录址
gii 存档 git add 添加文件内容到索引(无参数默认为所有文件)
gii 退档 git reset HEAD 回退添加的索引(无参数默认为所有文件)
gii 入库 git commit -m 提交变更到仓库(需要附带参数 版本说明)
gii 退库 git reset –hard 回退仓库到指定版本(需要附带参数 版本id)
gii 提交 git push 将当前内容提交到远程仓库
gii 新建分支 分支名 git branch 分支名 创建分支
gii 切换分支 分支名 git checkout 分支名 切换到指定分支
gii 删除分支 分支名 git branch -d 分支名 删除指定分支
gii 合并 仓库id git rebase -i 仓库id 合并版本库
gii 合并分支 分支名 git merge 分支名 将指定分支合并到当前分支
