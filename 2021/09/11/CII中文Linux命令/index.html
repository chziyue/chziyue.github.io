CII中文Linux命令

本文最后更新于：2024年10月11日 晚上

前言

CII 是基于 C++ 开发的 Linux 中文终端命令程序，安装后可以直接在终端使用中文命令来操作，大大的降低了 Linux 终端命令的操作门槛，摆脱了记不住英文命令的现象。

安装：

  • 进入 /usr/local/bin/ 目录：
  • 下载 CII:
  • 为 CII 添加可执行权限:

中英文命令对照表：

中文命令 英文命令 说明
cii 帮助 cii -h 显示帮助信息
cii 版本 cii -v 显示 CII 版本
cii 显示命令 cii -d 显示输出的英文命令
cii 隐藏命令 cii -a 隐藏输出的英文命令
cii 列表 ls -l 默认添加 -l 参数
cii 时间/日期 date 显示当前时间
cii 内核 uname -srm 显示系统内核版本
cii 处理器 cat /proc/cpuinfo丨grep name丨cut -f2 -d:丨uniq -c 显示 CPU 型号
cii 系统 cat /etc/os-release 显示系统信息
cii 位置 pwd 显示当前目录的位置
cii 关机 shutdown -h now 立即关机
cii 重启 shutdown -r now 立即重启
cii 进程 ps aux 显示所有进程
cii 进程 程序名 ps aux丨grep 程序名 显示单独程序进程，需要附带参数 (程序名)
cii 结束进程 进程PID kill -QUIT 进程PID 结束单独进程，需要附带参数 (程序PID)
cii 端口 netstat -lntp 显示监听的端口
cii 网卡 netstat -i 显示网卡信息
cii 大小 du -chs 显示文件或目录占用空间的大小
cii 新建目录 目录名 mkdir 目录名 需要附带参数 (目录名)
cii 新建文件 文件名 touch 文件名 需要附带参数 (文件名)
cii 下载 网址 wget 网址 需要附带参数 (网址)
cii 压缩 文件名.gz/.zip 目录/文件 tar cfz 或 zip -r 需要附带两个参数(压缩后文件名和要压缩的目录或文件)，自动识别压缩包名使用不同的命令来压缩，如需压缩 .zip 需要提前安装 unzip：cii 安装 unzip
cii 解压 文件名 tar -xvf 文件名 需要附带参数(文件名)，自动识别文件类型使用不同的命令来解压，如需解压 .zip 需要提前安装 unzip：cii 安装 unzip
cii 实时进程 top 实时显示所有进程 (键入 q 即可退出)
cii 在线用户 who 显示当前在线用户
cii 当前用户 whoami 显示当前用户
cii 主机名 hostname 显示主机名
cii 系统名 uname 显示系统名
cii 磁盘 df -h 显示磁盘占用的空间
cii 分区 fdisk -l 显示磁盘分区信息
cii 网络 ifconfig 显示当前网络信息 需要提前安装 ifconfig：cii 安装 net-tools
cii 清屏 clear 刷新屏幕，清除屏幕所有信息
cii 内存 free -m 显示当前内存信息
cii 可执行 chmod +x 文件名 该文件可执行
cii 不可执行 chmod -x 文件名 该文件不可执行
cii 可写 chmod 777 目录或文件名 该目录或文件可写
cii 不可写 chmod 755 目录或文件名 该目录或文件不可写
cii 全可写 chmod -R 777 目录或文件名 该目录及以下所有目录和文件可写
cii 全不可写 chmod -R 755 目录或文件名 该目录及以下所有目录和文件不可写
cii 复制 cp -r -f 目录或文件名 目标目录 复制目录或文件到目标目录下
cii 移动 mv -f 目录或文件名 目标目录 移动目录或文件到目标目录下
cii 配置源 自动配置国内源 Debian系列专用
cii 清理源 yum/apt/dnf 自动识别系统使用对应包管理工具
cii 更新源 yum/apt/zypper/dnf/pacman/apk 自动识别系统使用对应包管理工具
cii 更新包 yum/apt/zypper/dnf/pacman/apk/swupd 自动识别系统使用对应包管理工具
cii 搜索包 yum/apt/zypper/dnf/pacman/apk/swupd 自动识别系统使用对应包管理工具
cii 包信息 yum/apt/zypper/dnf/pacman/apk/swupd 自动识别系统使用对应包管理工具
cii 安装 yum/apt/zypper/dnf/pacman/apk/swupd 自动识别系统使用对应包管理工具
cii 卸载 yum/apt/zypper/dnf/pacman/apk/swupd 自动识别系统使用对应包管理工具
